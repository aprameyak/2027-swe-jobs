## How to Use

1. **Browse** — Click any of the quick links above to jump to the relevant job list.
2. **Search** — Use `Ctrl+F` / `Cmd+F` in your browser, or GitHub's search bar to filter by company, location, or role type.
3. **Filter** — Each table includes a `Remote` and `Sponsorship` column so you can quickly narrow down listings.
4. **Track** — Star this repository to get notified when new jobs are added.
5. **Contribute** — Found a listing we missed? [Open a PR](CONTRIBUTING.md) or [submit an issue](.github/ISSUE_TEMPLATE/add-job.yml).

## How to Contribute

Contributions are what keep this list alive and accurate. See [CONTRIBUTING.md](CONTRIBUTING.md) for full details.

**Quick guide:**

- **Add a job** → Edit the relevant markdown file and open a pull request.
- **Update status** → Change the status emoji in the `Status` column.
- **Fix a broken link** → Submit a PR with the corrected URL.
- **Report an issue** → [Open an issue](.github/ISSUE_TEMPLATE/add-job.yml) using the job submission template.

## FAQ

<details>
<summary><strong>How often is this updated?</strong></summary>

This list is updated manually as new job postings are found. Community contributions via pull requests are the primary driver of updates. Star the repo and watch for notifications.

</details>

<details>
<summary><strong>How do I know if a link is still active?</strong></summary>

Check the `Status` column. If a listing shows 🔴 Closed, the application window has passed. If a link appears broken, please [open an issue](https://github.com/aprameyakannan/2027-swe-jobs/issues) or submit a fix via PR.

</details>

<details>
<summary><strong>What counts as a "software engineering" role?</strong></summary>

This list focuses on roles with titles like Software Engineer (SWE), Software Development Engineer (SDE), Software Developer, and closely related technical roles (e.g., Platform, Infrastructure, Backend, Frontend, Full-Stack). Roles like Data Science, ML Engineering, Product Management, and DevOps are generally out of scope unless they are explicitly SWE-adjacent.

</details>

<details>
<summary><strong>Does this list include non-US roles?</strong></summary>

Yes. The `Location` column specifies the city and country (or "Remote"). Filter by location to find roles in your region.

</details>

<details>
<summary><strong>Can I submit a role at a small company?</strong></summary>

Absolutely. All company sizes are welcome as long as the posting is legitimate and publicly accessible.

</details>

<details>
<summary><strong>I found duplicate listings — what should I do?</strong></summary>

Please open a pull request removing the duplicate and keeping the more complete/accurate entry.

</details>
